# Vehicle Resale Business Intelligence Report (New-Wheels)

A quarterly business report built for a vehicle resale company experiencing declining sales.

## About
Imported and queried a multi-table relational database to answer key business questions
raised at CEO level. Identified root causes of declining sales and customer satisfaction,
and delivered data-driven recommendations.

## What's inside
- 10 SQL queries covering revenue, orders, customer ratings, shipping times, and discounts
- Query outputs with observations and insights for each
- Business metrics overview
- Executive recommendations

## Tech
SQL · MySQL · Window functions · CTEs · JOINs · Aggregate functions
